# Sistema de Gestión de Alquileres de Películas

Proyecto desarrollado con Spring Boot y PostgreSQL utilizando la base de datos Pagila.

## Requisitos
- Java JDK 17
- Maven
- PostgreSQL
- pgAdmin 4

## Pasos de instalación
1. Clonar el repositorio.
2. Abrir el proyecto en el IDE.
3. Configurar la base de datos en application.properties.
4. Ejecutar como Spring Boot App.

## Ejecución
El proyecto se ejecuta en http://localhost:8083

## Base de datos
Se utiliza la base de datos Pagila cargada en PostgreSQL.
